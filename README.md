# Proyecto: Precio del Oro en Euros

Este repositorio tiene como objetivo descargar el precio del oro en euros desde el año 2000, corregirlo utilizando el IPC (Índice de Precios al Consumidor) y generar tablas y gráficas para su análisis.

## Características principales
- Descarga automática del precio histórico del oro en euros.
- Corrección de los precios utilizando el IPC.
- Generación de tablas y gráficas para visualizar la evolución del precio real del oro.
- Implementaciones disponibles en **Python** y **R**.

## Estructura del repositorio
- Python/: Scripts y módulos en Python para la descarga, corrección y visualización de datos.
- R/: Scripts y archivos de proyecto en R para realizar las mismas tareas.
- Archivos CSV: Datos históricos de precios e IPC utilizados por ambos lenguajes.

## Uso
Consulta las carpetas Python/ y R/ para instrucciones específicas sobre cómo ejecutar los análisis en cada lenguaje.
